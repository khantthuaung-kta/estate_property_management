# Odoo Estate Property Management (Odoo 15)

This project is a **learning-based Odoo module** developed by following the official **Odoo documentation and tutorials** for **Estate Property Management** using **Odoo version 15**.

The purpose of this project is to understand:
- Odoo module structure
- ORM (models, fields, relations)
- Views (form, tree, kanban)
- Business logic and workflows
